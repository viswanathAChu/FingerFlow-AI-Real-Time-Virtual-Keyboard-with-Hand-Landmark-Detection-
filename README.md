# FingerFlow AI ⌨️
> Real-Time Virtual Keyboard with Hand Landmark Detection

## 📌 Overview
FingerFlow AI is a touchless virtual keyboard that uses your webcam and hand gestures to type — no physical keyboard needed.

## 🛠️ Tech Stack
- Python
- MediaPipe Hands
- OpenCV
- NumPy

## ⚙️ Installation
```bash
pip install opencv-python mediapipe numpy
```

## 🚀 Run
```bash
python virtual_keyboard.py
```

## 🖐️ How to Use
- Show your hand in front of the webcam
- Move your **index finger** to hover over keys
- **Pinch index + thumb** together to type
- Press **Q** to quit
